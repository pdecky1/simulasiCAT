# 📦 Langkah Instalasi

## 1. Clone repositori  
```bash
git clone <YOUR_GIT_URL>
```

## 2. Masuk ke direktori proyek  
```bash
cd <YOUR_PROJECT_NAME>
```

## 3. Instal semua dependensi  
```bash
npm install
```

## 4. Jalankan server pengembangan  
```bash
npm run dev
```
